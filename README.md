# Password Generator Project

## Description
This is a simple Python console-based Password Generator.  
It generates a random password based on the length entered by the user.

## Features
- User-defined password length
- Random password generation
- Input validation

## Concepts Used
- secrets module
- string module
- input() function
- string manipulation
- try/except handling

## How it works
User enters password length and program generates a random secure password using letters and numbers.
